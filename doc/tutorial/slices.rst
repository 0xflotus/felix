Slices
======

A slice is a range of integers.

Type
----

The type is defined in the library as

.. code-block:: felix

    slice[int]
 

Inclusive Slice
---------------

From first to last, inclusive:

.. code-block:: felix

    first..last

Exclusive Slice
---------------

From first to last, exclusing last:

.. code-block:: felix

    first..<last
